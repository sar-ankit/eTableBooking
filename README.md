# Welcome To eTableBooking
The eTableBooking front-end web application is a user-friendly interface for customers to search and book tables at restaurants, cafes, and other establishments. The application is built using React, and is designed to provide a seamless user experience also user can see the menu of restaurants and services avaliable in restaurants.
